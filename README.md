# DCosta Ordenhadeiras - Front-end

Interface web do e-commerce DCosta Ordenhadeiras.

## Tecnologias

- React
- TypeScript
- Vite
- TailwindCSS
- Lucide React
- Context API

---

## Funcionalidades

- Catálogo de produtos
- Filtro por categorias
- Carrinho de compras
- Checkout integrado
- Integração com Mercado Pago
- Login e cadastro
- Área do cliente
- Histórico de pedidos
- Retry de pagamento
- Painel administrativo
- Responsividade mobile

---

## Instalação

Clone o projeto:

```bash
git clone <url-do-repositorio>

Entre na pasta:

cd front-end

Instale as dependências:

npm install
Variáveis de ambiente

Crie um arquivo .env

VITE_API_URL=http://localhost:3001
Rodando o projeto
npm run dev
Build produção
npm run build
